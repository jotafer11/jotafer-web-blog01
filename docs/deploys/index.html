          # Netlify

          1. con el proyecto listo, ejecutar en la carpeta vue
          npm run build
          1. netlify, Sites, Deploy Manually (arrastrar carpeta)

          # Firebase

          1. crear proyecto en firebase e instalar modulo firebase con
          npm install firebase @angular/fire --save
          1. crar firestore db en firebase, configurar, importar en app.module
          import { AngularFireModule } from '@angular/fire';
import { environment } from '../environments/environment'


imports: [

    AngularFireModule.initializeApp(environment.firebaseConfig)
    
],
          1. importar servicios de firestore en app.module y en el servicio.
          import { AngularFirestoreModule } from '@angular/fire/firestore';

          # VuePress

          1. iniciar servidor
          yarn dev
          1. generar carpeta dist
          yarn build
          1. deploy a github pages
